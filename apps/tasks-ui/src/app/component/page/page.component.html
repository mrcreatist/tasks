Welcome to Tasks

  • You can create board by clicking on button from the top bar
  • In this web app you can arrange tasks in 2 ways
    1. By Created - Sorting on the basis of creation date
    2. Free Fall - Place task anywhere within the boards
  • You can update the setting by clicking on the settings icon
  • This project has the capability to sync the boards over server and for that socket has been used but you can ignore the code since it was out of requirement
  • This project is mobile responsive
